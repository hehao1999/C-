Welcome To Use MyCAD

MyCAD is a very unprofessional very little program for drawing maps.You can use it for fun.You can only get happiness from it, but don't take it seriously.

Let us step into the topic.What can this software do? Let's see:

FUNCTIONS

  • Import file

    Import files to generate simple maps,just a simple map!!!You should click File,import to achieve this function.But unfortunately, the imported file should use the format we specified, and it is not free to draw the map. For details, please refer to "TestData.csv".

  • Export image

    By doing this, you can export your map to 'jpg' image or 'bmp' image.

  • Buttons

    You should try them yourself.And I am a greenhand, there must be a lot of bugs and deficiency.
